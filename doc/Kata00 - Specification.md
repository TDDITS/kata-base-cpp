# Kata 0 - Yourself

## Tasks

1. Store yourself in an object
2. Store others too
3. Form groups of people
4. List groups and people with full name ordered by age

### Addition

* Person details:
    * first name
    * last name
    * age
* Group details:
    * name
    * size (calculated: 1-5 small, 5-20 medium, 20+ large)
